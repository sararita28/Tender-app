# Tender-app

<img align="right" height="100px" src="logo.png"/>

<b>Tender</b> is a Tinder web-app alternative that allows users to swipe (i.e. pick matches) based on other users' personalities (or rather descriptions) instead of their pictures. Only when two users have matched (i.e. they have both swiped right) can they see the other person's picture.

<img width="100%"  src="ezgif.com-gif-maker (6).gif" />

### Stack Used: 
This app was built using the MERN stack (MongoDB, Express, React, NodeJs) as well as bcrypt, cors, mongoose, multer, axios and other dependencies.

### Scope
The code is divided into 2 folders; one for the frontend named <b>frontend</b> and one for the <b>backend</b>.

### Running the code:
``` 
git clone https://github.com/sararita28/Tender-app.git 
npm i
npm start
```

### User Flow:
Users can:
- Register 
- Login
- Edit their profile
- Send and receive messages
- View people's descriptions
